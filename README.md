animabf-sandbox

Mundo de **Anima Beyond Fantasy Core exxet 1.0** para FoundryVTT, creada con Sandbox.

Para utilizar este mundo es necesario el sistema [Sandbox](https://gitlab.com/rolnl/sandbox-system-builder/-/tree/master) de FoundryVTT.

Manifest (instalar mundo): [manifest](https://raw.githubusercontent.com/Luvero-1/animabf-sandbox/main/world.json).
